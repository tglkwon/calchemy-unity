# 콘텐츠 데이터 명세 (Content Spec)

## 1. 카드 (Cards)
- **ID/Name/Grade**: 기본 식별 정보.
- **Single Logic**: 낱개 발동 시 로직 (YAML/JSON).
- **Bingo Logic**: 빙고 포함 시 추가 로직.

## 2. 유물 (Relics)
- 전투 중 지속적으로 영향을 미치는 패시브 아이템.
- **Trigger**: 발동 시점 (예: ON_BINGO, ON_TURN_START).

## 3. 포션 및 소비템 (Potions)
- 전투 중 즉석에서 사용하는 일회성 아이템.
- 카드 속성 변경이나 위치 교환 등 전략적 변수 창출.

## 4. 이벤트 (Events)
- 맵 진행 중 발생하는 선택형 서사.
- 선택지에 따른 보상 또는 패널티 부여.
