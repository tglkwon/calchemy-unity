# 그리드 및 빙고 (Grid & Bingo)

## 1. 빙고 효과 (Bingo Effects)
한 줄(가로, 세로, 대각선)을 완성했을 때 발생하는 강력한 효과입니다.

- **🔥 불 빙고**: 무작위 적에게 강력한 추가 피해.
- **🌱 흙 빙고**: 골렘에게 대량의 방어도 부여.
- **💧 물 빙고**: 골렘 체력 회복.
- **🍃 바람 빙고**: 골렘의 공격력 영구 버프.
- **🌈 조화 빙고**: 적 전체 피해 + 골렘 회복/방어 복합 효과 (4원소 모두 포함 시).

## 2. 그리드 조작 (Grid Manipulation)
카드의 효과 중 그리드 자체를 변화시키는 기믹입니다.

- **TRANSFORM**: 특정 카드의 원소 속성을 변환.
- **SWAP**: 두 카드의 위치를 교환.
- **REPLACE**: 카드를 제거하고 새로운 카드로 대체.
- **UPGRADE**: 카드의 등급을 강화.

### 타겟팅 규칙
- **방향형**: UP, DOWN, LEFT, RIGHT (벽 반사 로직 포함).
- **범위형**: NEAR_4 (십자), NEAR_8 (주선 8칸).
- **전역형**: RANDOM, ALL.
