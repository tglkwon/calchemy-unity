---
description: 실행 중인 유니티 MCP 서버의 상태를 확인하는 워크플로우
---

유니티 MCP 서버가 현재 활성화되어 있고 응답하는지 확인합니다.

### 1단계: 포트 점유 확인
// turbo
1. 8080 포트를 사용 중인 프로세스가 있는지 확인합니다.
   `netstat -ano | findstr :8080`

### 2단계: HTTP 응답 확인 (Health Check)
// turbo
1. MCP 엔드포인트에 접속을 시도하여 상태 코드를 확인합니다.
   `curl -o /dev/null -s -w "%{http_code}" http://127.0.0.1:8080/mcp`

### 3단계: 결과 보고
- 포트가 점유되어 있고 HTTP 200(또는 성공 응답)이 오면:
  - "✅ 유니티 MCP 서버가 활성화되어 있습니다 (http://127.0.0.1:8080/mcp)."
- 포트는 있지만 응답이 없으면:
  - "⚠️ 포트는 열려 있으나 서버가 응답하지 않습니다. 재시작(/connect-unity-mcp)을 권장합니다."
- 포트가 비어 있으면:
  - "❌ 실행 중인 유니티 MCP 서버가 없습니다."
